# Getting Started with Create React App

# recruitment-task
React/TS list project.

## Installation

1. Clone this repo

```sh
git clone https://github.com/1991Pawel/zadanie-rekrutacyjne.git
```

2. Install NPM packages

```sh
npm install 
```
3. Start a development server
npm 

```sh
npm run start
```
